# AI-Based Personalized Book Recommendation System

## Project Description
A web application that uses Machine Learning 
and NLP to recommend personalized books based 
on user preferences and analyzes review 
sentiments to predict whether a user will 
enjoy a book.

## Tech Stack
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: Python Flask
- Database: MySQL
- ML: Scikit-learn
- NLP: NLTK + TextBlob
